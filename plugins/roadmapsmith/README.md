# RoadmapSmith Codex Plugin

AI-assisted ROADMAP.md management with evidence-backed task validation.

## Skills

| Slash command | Purpose |
|---|---|
| `/roadmap` | Native slash palette and feature discovery |
| `/roadmap-zero` | Guided one-command setup flow |
| `/roadmap-maintain` | Sync, validate, and maintain the roadmap |
| `/roadmap-status` | Inspect readiness and health |
| `/roadmap-init` | Initialize a new roadmap |
| `/roadmap-generate` | Regenerate from codebase scan |
| `/roadmap-validate` | Validate task-to-code evidence |
| `/roadmap-update` | Update task checked state |
| `/roadmap-audit` | Audit task alignment |
| `/roadmap-setup` | Generate host integration files |
